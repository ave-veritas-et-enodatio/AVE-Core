## Predicted Thrust Profile (Linear Base)

Because the acoustic shockwave mechanism mathematically inverts into a rarefaction wave under the correct structural denominator operator, the purely nonlinear excess thrust cannot be predicted by standard compressive fluid dynamics. The table below represents only the linear cross-term amplifications as a function of the local permittivity modulation ($\varepsilon_{eff}/\varepsilon_0$).

| $V_{DC}$ (kV) | Cross-Term $F_{lin}$ ($\mu$N) | $C_{eff}/C_0 = 1/S$ |
|---|---|---|
| 0 | 0 | 1.000 |
| 10 | 156 | 1.027 |
| 20 | 313 | 1.125 |
| 30 | 469 | 1.377 |
| 40 | 626 | 2.493 |
| 43 | 673 | 5.923 |

The linear cross-term currently serves as the only mathematically sound bound for macroscopic thrust generation, pending a rigorous re-derivation of the macroscopic rarefaction wave integral using the Maxwell Stress Tensor.
