# Ch.9 Antimatter Annihilation and Parity Inversion

Matter-antimatter annihilation is modelled as the deterministic mechanical collision of two counter-rotating topological flywheels. The positron is the parity-inverted ($-\boldsymbol{\omega}$) unknot of the electron. When opposing vorticities cancel ($\boldsymbol{\omega} + (-\boldsymbol{\omega}) = 0$), the topological boundary snaps, releasing $E_{knot} = \frac{1}{2}I\omega^2$ as propagating radiation. Pair production is the inverse process: volumetric wave shear above $V_{yield}$ fractures a linear wave into two contra-rotating vortex dipoles.

## Key Results

| Result | Statement |
|---|---|
| Annihilation mechanism | Vorticity cancellation: $\boldsymbol{\omega} + (-\boldsymbol{\omega}) = 0$ $\to$ boundary snap |
| Rotational energy | $E_{knot} = \frac{1}{2}I\omega^2$ |
| Pair production threshold | $E_\gamma \geq 2m_e c^2 = 1.022$ MeV |
| Soliton rest mass | $E = mc^2 = \frac{1}{2}LI^2 + \frac{1}{2}CV^2$ |
