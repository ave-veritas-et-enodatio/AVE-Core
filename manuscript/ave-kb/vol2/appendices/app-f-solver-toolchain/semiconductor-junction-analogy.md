## The Semiconductor Junction Analogy

The saturation shell is the gravitational analogue of a semiconductor p--n junction depletion region. This analogy provides a systematic framework for extracting all "datasheet" parameters.

### Small-Signal vs. Large-Signal Analysis

| Regime | Semiconductor | Black Hole (AVE) |
|---|---|---|
| **Small-signal** (linear) | Linearised about DC bias; $g_m$, $r_\pi$, $C_\pi$ | Linearised about static BH; QNM $\omega_R$, $Q$, radiation resistance |
| **Large-signal** (nonlinear) | Switching, saturation, turn-on transient | Merger ring-up, Axiom 4 saturation onset |
| **Breakdown** | Avalanche at $V_{BR}$; carrier multiplication | Dielectric rupture at $\varepsilon_{11} = 1$; topology melting |

### BH Transistor Datasheet

| Parameter | Symbol | Formula | Source |
|---|---|---|---|
| Breakdown voltage | $V_{BR}$ | $V_{\mathrm{SNAP}}$ | Axiom 4 |
| Junction frequency | $f_0$ | $\ell c / (2\pi r_{\mathrm{eff}})$ | Orbital mode |
| Transit time | $\tau_{\mathrm{tr}}$ | $W/c$ | Depletion width |
| Bandwidth | BW | $f_0 / Q$ | Radiation loss |
| Rise time | $t_r$ | $2.2 / (2\pi f_0)$ | RC time constant |
| Fall time (ringdown) | $\tau$ | $Q / (\pi f_0)$ | QNM decay |
| Depletion width | $W$ | $5\,M_g$ | $r_{\mathrm{sat}} - r_s$ |
| Hawking temperature | $T_H$ | $\hbar c^3/(8\pi k_B G M)$ | Quantum |
