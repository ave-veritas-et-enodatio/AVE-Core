# Ch. 8: The Planck Scale and String Theory

Chapter 8 resolves the string theory dimensionality crisis by providing the 3D continuous non-linear LC metric that stabilizes 1D flux tubes without requiring unobservable extra dimensions, and derives the empirical string tension and Regge slope from the baseline electron unknot.

## Key Results

| Result | Statement |
|---|---|
| AVE string tension | $T_{AVE} = m_e^2 c^3/\hbar \approx 0.212$ N |
| Hadronic Regge slope | $\alpha' = \hbar/(2\pi m_e^2 c^3) \approx 0.750$ GeV$^{-2}$ (17% of empirical $0.9$ GeV$^{-2}$) |
| QCD string tension | $\alpha'_{baryon} = \ell_{node}/(2\pi m_p c^2) \approx 4.09 \times 10^{-4}$ GeV$^{-2}$ |
| Extra dimensions | Unnecessary: volume-bearing flux tubes + dielectric saturation + topological stability in 3+1D |
