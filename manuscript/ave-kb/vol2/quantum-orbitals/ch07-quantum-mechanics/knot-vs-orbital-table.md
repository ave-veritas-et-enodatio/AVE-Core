## Table: Knot Topology vs Orbital Geometry --- Scale Separation

| **Property** | **Knot (particle)** | **Orbit (mode)** |
|---|---|---|
| Axioms | 1 (lattice) + 3 (topology) | 2 (strain, Poisson) |
| Scale | $\rho = 1/(2\pi)\;\ell_{\text{node}}$ | $R = n^2 a_0 / Z$ |
| Typical size | $\sim 0.16\;\ell_{\text{node}}$ | $\sim 100$--$200\;\ell_{\text{node}}$ |
| Circuit element | Inductance $L$, mutual $M$ | Capacitance $C$, mutual $C_m$ |
| AVE operator | Op1 (Neumann integral) | Op4 (Coulomb $1/r$) |
| Coupling formula | $k = M/\sqrt{L_1 L_2}$ | $\sigma_{\text{Hopf}} = 1/2$ |
| $m_l$ dependent? | **Yes** ($M > 0$ vs $M = 0$) | No (Gauss, isotropic) |
| IE effect | Bonding mode shift | $Z_{\text{eff}}$ screening |
| Loss mechanism | Op10 (junction projection) | None (screening is lossless) |
