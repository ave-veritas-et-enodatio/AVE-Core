# Alzheimer's Disease as Progressive Impedance Drift

Demyelination changes the characteristic impedance of axonal transmission lines by modifying $\varepsilon_\text{axon}$ (myelin is dielectric insulation). In the TL framework:

1. Myelin loss increases $C_m$ (thinner dielectric $\to$ higher capacitance), lowering $Z_\text{axon}$.
2. This detunes the impedance matching at every downstream synapse, increasing $|\Gamma|$ at all connected junctions.
3. Standing wave patterns that depend on the detuned segments lose coherence (memory loss).
4. Recently formed memories are lost first because their impedance-matched pathways have had less time to develop structural reinforcement (myelination thickening).

## Falsifiable Prediction

Regional myelin integrity (measurable via diffusion tensor MRI) should correlate with memory test performance in exactly the pattern predicted by the network $S_{11}$: the memory domains served by the most demyelinated tracts should fail first, independent of amyloid plaque location.
