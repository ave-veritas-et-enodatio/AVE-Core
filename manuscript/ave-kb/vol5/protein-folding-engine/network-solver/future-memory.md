# Memory as Topological Impedance Locking

## Core Hypothesis

Long-term potentiation (LTP) --- the cellular basis of memory --- may be mechanically equivalent to **impedance matching** at a synaptic junction. When a synaptic connection strengthens ($\Gamma \to 0$), the junction becomes transparent: standing wave patterns can propagate through it with minimal reflection. A specific memory is a **topologically locked standing wave pattern** sustained by a network of impedance-matched boundaries. Forgetting is impedance drift: weakening synapses increase $|\Gamma|$, and the standing wave pattern decays.

## Falsifiable Prediction

Memory capacity scales with the number of independent standing wave modes that the network can sustain --- a topological invariant (eigenvalue count of the network $S^\dagger S$ matrix), not a synaptic weight count.
