# Tensor Plates: Passive Matrix Multiplication

Macroscopic Tensor operations map identically onto passive spatial geometric matrices. Refractive impedance structuring inherently modulates continuous propagation via inverse scaling limits bounding the ALU geometry.

The Tensor Plate is a planar dielectric slab with a spatially varying permittivity pattern $\kappa(\mathbf{r})$ engineered to implement a specific linear transformation. When a multi-harmonic wave packet (the GISA instruction) enters the Plate, each spectral component diffracts according to its frequency and the local grating pitch. The output spatial distribution is the matrix product of the input spectrum and the Plate's transfer function.

This is inherently lossless (within the linear Regime I operating domain) and requires zero clock cycles — the multiplication completes in a single wave transit time across the Plate.
